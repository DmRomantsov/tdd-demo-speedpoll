# Demo project with Test-driven development (TDD) approach 

## Requirements: 
- Java 11

## Business requirements: 
* As a user I want to provide my address and internet speed in Mb\sec and retrieve average internet speed by this address. 
## How to

### Run tests
```shell script
./gradlew test
````

### Start service
```shell script
./gradlew bootRun
````
